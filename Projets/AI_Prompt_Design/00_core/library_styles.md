# Library of styles

Bibliothèque de **briques stylistiques réutilisables**. Chaque entrée =
nom court FR + formulation EN éprouvée + IA(s) où ça marche bien.

> **Usage** : quand Mickael évoque un style nommé (*"à la Wes Anderson"*,
> *"flat design"*), Jarvis pioche ici la formulation EN testée plutôt
> que d'inventer.

## Procédure d'ajout

1. Itération converge avec une formulation EN qui marche.
2. Jarvis propose l'ajout : nom FR + EN + IA validées.
3. Mickael valide → entrée ajoutée dans le tableau correspondant.
4. Si la même brique sert sur 3+ itérations : la promouvoir en
   "consolidée" (gras dans le tableau).

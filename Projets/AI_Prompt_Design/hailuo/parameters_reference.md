# Hailuo 02 — référence paramètres

## Paramètres Hailuo 02

| Param | Valeurs | Effet |
|---|---|---|
| Duration | 6 s, 10 s | 6 s plus stable |
| Resolution | 720p, 1080p | 1080p disponible |
| Aspect ratio | 16:9, 9:16, 1:1 | — |
| Image input | upload | Image-to-video |
| Audio | on / off | Audio basique (ambient music) |
| Negative prompt | mots-clés | Anti-éléments |
| Prompt language | EN recommandé | CN auto-traduit moins bon |

## Tarification (mai 2026)

Hailuo se positionne comme **rapport qualité/prix le plus compétitif du
marché**. Voir `pricing_vs_quality.md` pour comparatif chiffré vs Pika /
Runway Turbo / Luma Flash.

## Combinaisons typiques Mickael

### Itération rapide budget-friendly
```
Hailuo 02, 6s, 1080p, 16:9, audio: off
```

### Avec audio ambient music
```
Hailuo 02, 6s, 1080p, audio: on, prompt: "Audio: <mood>"
```

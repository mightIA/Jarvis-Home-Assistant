# HA — Raccourcis clavier

Toggle « Raccourcis clavier » **ON** dans le profil utilisateur Mickael.

| Raccourci | Action | Note AZERTY FR |
|---|---|---|
| `Shift + ?` | Afficher panneau raccourcis | ⚠️ Obtenu physiquement par `Shift+,` (touche qui produit `?`) |
| `A` | Ouvrir Assist | |
| `C` | Créer | |
| `E` | Entités | |
| `D` | Appareils | |
| `M` | Ma barre latérale | |
| `Ctrl+K` | Recherche rapide | |

## Piège AZERTY

Sur clavier français, `Shift+?` n'est pas accessible directement (la touche
`?` est en `Shift+,` sur AZERTY). C'est documenté dans HA mais le piège
revient régulièrement.

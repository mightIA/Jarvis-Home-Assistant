# 90_Archives/

**Ce qu'on range ici** : tout projet clos ou note devenue obsolète mais
qu'on ne supprime pas (utile pour relire l'historique des décisions).

## Convention

- **Projets** : dossier `<AAAA-MM>_<NomProjet>/` (ex. `2026-04_Mode_Reactif_Phase1/`)
- **Notes obsolètes** : ajouter `archived` au frontmatter `tags` + déplacer
  ici. Ne **jamais** modifier le contenu, juste le déplacement.

## Règle d'or

Une note archivée **ne se réécrit pas**. Si l'info devient à jour ailleurs,
créer une nouvelle note dans `10_Domaines/` ou `20_Projets/` qui cite
l'archive en référence.

C'est la même règle que pour `memory/historique/` à la racine du projet
(cf auto-memory `feedback_projets_vs_runtime.md` S33).

## Pruning

Pas de purge automatique. Une fois par an (rotation 1er janvier, cf
section 9 `CLAUDE.md`), zipper les archives de l'année dans
`Archives/wiki_archives_AAAA.zip` à la racine du projet si volume
significatif.
